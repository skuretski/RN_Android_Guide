How-To-Guide for React Native Android

This guide is intended to help mobile developers get started with React Native for Android.


With its use of JSX and JavaScript, React Native greatly helps cross platform development since you don't have to strictly code Objective-C, Swift, or Java. Of course, there are still some differences between Android and iOS development.

While React Native for iOS was released in early 2015, React Native for Android was released late 2015. On the whole, React Native is still relatively new, and changes are happening all the time. For the most up-to-date information, check out the React Native Docs or React Native GitHub.

Baseline knowledge

Since this guide isn't intended to teach coding from the ground up, there is some expected baseline knowledge.

  • Node.js and NPM
  • HTML, CSS, and/or JSX
  • JavaScript
